# Steam Achievement Viewer

A lightweight C++ console utility using the Steamworks SDK to extract and export your local achievement completion histories into structured JSON data schemas.

## How to Run (Pre-built)
1. Download the latest compiled package from the **Releases** tab.
2. Ensure your desktop **Steam Client** is running and logged into your profile.
4. Run `SteamAchievementViewer.exe`**HINT:** You'll need to know your game's APPID 

## How to Build from Source
* Built using **Visual Studio** (Desktop Development with C++ workload).
* Requires the **Steamworks SDK** headers and library bins linked into your solution properties.
